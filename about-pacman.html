3D Pac-Man Project

About

I created a 3D version of Pac-Man for my A-Level Computer Science coursework. This project gained me full marks, giving me an A* in this coursework.

Project Info

Role: Games Programmer
Team Size: 1
Time frame: 6 months
Engine: Unity (C#)

What I learnt

In this project I dived into creating NavMeshes for the AI ghosts to naviagte to follow the main character. The navigated using Dijkstra's algorithm to find the shortest path to the player. This taught me a lot about NavMeshes and AI behaviour. I also developed a menu/settings system which allowed the player to changed audio and screen size settings.
