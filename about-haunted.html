Christmas Escape Room

About

Unity app to supplement the work Christmas Escape room. This was a blend between physical experiments and puzzles on the application.

Project Info

Role: Games Developer
Team Size: 1
Time frame: 1 month
Engine: Unity (C#)

Introduction

In the first two months of my new job I was tasked with creating an interactive escape room for the work christmas party. This consisted of 6 challenges which require the user to complete the experiments to reveal numbers and words to be inputted into the application. There is a sequence lock room and a NYT Connections style room on the application. Play through the application below, as you will not have the physical experiments I will give you the inputs you need. Challenge 1 - Blue, Yellow, Red. Challenge 2 - 0, Challenge 3 - 8, Challenge 4 - You need to workout these connections, Challenge 5 - (d, c, b, b, c) (3), Challenge 6 - You need to work this out.

What I learnt

The main challenging code for this application was trying to replicate the NYT Connections puzzle. I tried a few different approaches with this code but settled on using dictionarys for grouping the connections. Another main part of this application was the audio manager, I created a prefab to be dropped in each level which you could feed custom sound clips into for each state like correct, incorrect, refresh etc.
